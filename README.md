# m-gpux
A CLI hub for managing Modal accounts, billing links, and connecting to Modal GPUs seamlessly with Interactive Notebooks and Terminals.

## Installation

```bash
pip install -e .
```

## Usage

```bash
m-gpux --help
m-gpux hub       # Interactive GPU allocation and connection
m-gpux account   # Manage different Modal profiles
m-gpux billing   # Open usage and billing
```
